## Aula 1

- [x] iniciando o projeto

  - [x] types express -> `yarn add @types/express -D`

  - [x] criando arquivo de conf. typescript -> `yarn tsc --init`
  - [x] intalando tradutor para o node -> `yarn add ts-node-dev -D`
